# assets/

此目录存放应用静态资源。

## app_icon.ico

应用程序图标文件。

- **格式**：ICO（多尺寸，推荐包含 16×16、32×32、48×48、256×256）
- **生成**：首次构建时由 `packaging/create_icon.py` 自动生成占位图标
- **替换**：社区成员可替换为正式设计的图标

### 制作指南

推荐使用以下工具制作正式图标：
- **GIMP**（免费）：导出为 ICO 格式时勾选多尺寸
- **IcoFX**（Windows）：专业 ICO 编辑器
- **在线工具**：https://convertico.com/

图标设计建议参考群星游戏的星系/UI 风格（深色背景 + 蓝色星芒）。
